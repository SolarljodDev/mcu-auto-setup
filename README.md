# mcu-auto-setup
One-script MCU dev setup for VS Code. Auto-detects or downloads GCC, CMake, Ninja and OpenOCD. Supports STM32 (ARM Cortex-M) and WCH CH32V (RISC-V). No specialized IDE required. | Автонастройка окружения для МК в VS Code — находит или скачивает GCC, CMake, Ninja и OpenOCD. STM32 и WCH CH32V из коробки.
